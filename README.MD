# Quizme


A CLI tool that will help to test knowledge.



```
quizme
```


- Ask a question
- Evaluate answer or have me self-evaluate
- Log the results



```
sh
quizme --add
```

- Start a dialogue
- Ask me for a question
- Ask me for the answer


## What do I need?

- Parse arguments
- Interactive command line
- Persist my data - using JSON (ATM)
   - Reading/Writting from the file system
- Self-evaluating or auto-evaluating
